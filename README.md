# GTuner
GTuner is a guitar tuner based on Python.
![](./GTuner.png)
## The following python libraries should be installed to use GTunner
- pyaudio
- numpy
- scipy
- matplotlib

## Usage
Use
```
python main
```
to run, and press `q` to quite the program.
